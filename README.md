# OWASP-Juice-Shop-Security-Lab
Beginner cybersecurity lab using Ubuntu, Docker, and OWASP Juice Shop to explore web application security concepts like authentication behavior, input validation, and access control in a controlled environment.

Web Security Lab – OWASP Juice Shop:
 Overview:-

This project is a beginner-friendly cybersecurity lab where I explored basic web application security concepts using Ubuntu, Docker, and OWASP Juice Shop. The goal was to understand how modern web applications behave and how common security issues can be identified in a safe, local environment.

 Tools & Technologies:
Ubuntu Linux
Docker
OWASP Juice Shop

 What I Learned:
How to set up and run a web application locally using Docker
Basics of web application structure and user interactions
Understanding how authentication and input fields behave
Introduction to access control and hidden routes in web apps
Observing how applications handle different types of input

 Project Evidence:

All screenshots related to setup, testing, and exploration are included in the /screenshots folder.

 Ethical Note:

All testing and exploration were done in a local intentionally vulnerable environment (OWASP Juice Shop) for learning purposes only. No real-world systems were accessed or affected.

 Conclusion

This project helped me take my first step into cybersecurity by understanding how web applications work from a security perspective. I’ll continue building on this foundation by exploring more structured security testing concepts.
